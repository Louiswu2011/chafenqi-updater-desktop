# 中二节奏/舞萌DX传分器

Compose Multiplatform实现的传分工具，支持Windows系统。

[iOS端](https://github.com/Louiswu2011/chafenqi)

[安卓端](https://github.com/Louiswu2011/chafenqi-android)

## 主要功能
- [x] 舞萌DX/中二节奏数据上传
- [x] 设置/修改QQ绑定
- [x] 设置/修改水鱼网账号绑定
- [x] 数据同步上传到水鱼网

## 使用方法
### QQ群（推荐）
请到双端任一Q群内获取最新版本安装包。
### Github Release（更新较慢，不推荐）
1. 下载Release中的msi文件
2. 安装
3. 开始传分

## 编译项目
1. Clone到本地
2. Intellij IDEA打开编译
